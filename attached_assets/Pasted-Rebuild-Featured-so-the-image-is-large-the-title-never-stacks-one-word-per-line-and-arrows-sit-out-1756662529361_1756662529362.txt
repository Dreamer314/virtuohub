Why this will stop the nonsense

The grid uses explicit minmax columns, so the text column cannot collapse and force vertical words.

The headline shrinks with the column using cqi, so you never touch copy.

The section has its own resets inside #featuredV2, so random global typography or prose styles cannot override it.

Arrows live in a separate overlay that spans the section, so they do not sit on top of the image.
