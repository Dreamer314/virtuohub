**What this does**
- It only applies in light theme (`[data-theme="light"]`).
- Adds visible borders to inputs and pills.
- Forces dropdown menu text to dark on white.
- Gives clear color pairs for success/info/price chips with AA contrast.

If your pills already have helper classes (like `vh-pill--price`, `status-pill--active`) this will style them. If not, the generic `.vh-pill/.vh-badge/.vh-chip` line will still improve their base contrast. It’s intentionally gentle to avoid side effects.
